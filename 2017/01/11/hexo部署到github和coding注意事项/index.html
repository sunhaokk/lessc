hexo部署到github和coding注意事项

  1. 在github新建项目的时候一定输入的是youname.github.io其中youname是你的用户名。
    因为github默认的页面，就是你的用户名。所以这个不要乱改。

  2. 同理coding在新建项目的时候一定也要是youname。你的用户名。coding默认也是用户名即是主页，否则生成的页面地址是http://youname_.coding.me/youproject。

  3. 在hexo的配置文件_config.yml中，因为yml要求规范严格，所以一定要注意缩进和空格。
